# 🎬 MANEMA – Movie Ticket Booking System

A modern web application built for booking movie tickets online.
Developed as a project for the **KHEC Project Defense**.

---

## 🚀 Overview

**MANEMA** is a full-stack web application that provides an interactive and efficient movie ticket booking experience. It combines a dynamic frontend with a powerful backend to deliver smooth performance, modern UI, and secure authentication.

---

## ✨ Features

* 🎟️ Browse movies and showtimes
* 🪑 Select seats interactively
* 🔐 Secure user authentication (JWT-based)
* ⚡ Fast and responsive UI
* 🎨 Smooth animations for better UX
* 🌐 RESTful API integration

---

## 🛠️ Technologies Used

### 🎨 Frontend

| Technology   | Purpose                                |
| ------------ | -------------------------------------- |
| React.js     | Building dynamic user interfaces       |
| Axios        | Handling HTTP requests to backend APIs |
| GSAP         | Creating smooth and modern animations  |
| Tailwind CSS | Responsive and utility-first styling   |

---

### ⚙️ Backend

| Technology            | Purpose                              |
| --------------------- | ------------------------------------ |
| Django                | Backend web framework                |
| Django REST Framework | Building RESTful APIs                |
| DRF Simple JWT        | Authentication using JSON Web Tokens |
| Django CORS Headers   | Handling cross-origin requests       |

## ⚙️ Installation & Setup

### 🔹 Clone the Repository

```bash
git clone https://github.com/your-username/manema.git
cd manema
```

### 🔹 Backend Setup

```bash
cd backend
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

### 🔹 Frontend Setup

```bash
cd frontend
npm install
npm start
```
